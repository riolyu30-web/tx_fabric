# 产品图片目录

此目录用于存放所有产品的图片文件。

## 图片命名规范

**格式：** `产品编号_图片序号.jpg`

**示例：**
- `801_1.jpg` - 801#涤天丝（曲天丝）主图
- `801_2.jpg` - 801#涤天丝（曲天丝）第2张图
- `802_1.jpg` - 802#天丝仙女条主图
- `901_1.jpg` - 901#2060洗水竹节主图

## 图片要求

### 文件格式
- 推荐：JPG/JPEG
- 备选：WebP（更小体积，更好性能）

### 图片尺寸
- 最小：800x800px
- 推荐：1200x1200px
- 最大：2000x2000px
- 宽高比：1:1（正方形）

### 文件大小
- 单张图片不超过 500KB
- 建议使用图片压缩工具优化

## 使用方法

### 1. 添加新产品图片

1. 按照命名规范准备图片（如：`815_1.jpg`, `815_2.jpg`）
2. 将图片复制到此目录
3. 运行同步脚本：
   ```bash
   node scripts/manage-images.js sync
   ```

### 2. 更换现有图片

1. 找到对应产品编号的图片文件
2. 用新图片替换旧文件（保持文件名不变）
3. 刷新网站即可看到更新

### 3. 批量添加图片

1. 将所有产品图片按命名规范放入此目录
2. 运行同步脚本自动更新 `data/products.json`
   ```bash
   node scripts/manage-images.js sync
   ```

## 产品编号对照

| 系列 | 编号范围 | 数量 |
|------|---------|------|
| 天丝系列 | 801# - 832# | 28个 |
| 绣花底布系列 | 601# - 606# | 6个 |
| 棉布系列 | 901# - 914# | 13个 |

## 注意事项

- 文件名中的产品编号**不要包含 `#` 符号**（如：`801_1.jpg` 而非 `801#_1.jpg`）
- 每个产品至少需要1张图片（`产品编号_1.jpg`）
- 最多支持5张图片（`_1.jpg` 到 `_5.jpg`）
- 图片按序号顺序显示，第1张为主图
- 确保图片背景干净整洁，推荐纯白色或浅色背景

## 管理工具

查看详细的图片管理指南：`docs/图片管理指南.md`

使用图片管理脚本：
```bash
# 查看帮助
node scripts/manage-images.js

# 同步本地图片到JSON
node scripts/manage-images.js sync

# 列出所有产品图片使用情况
node scripts/manage-images.js list
```
