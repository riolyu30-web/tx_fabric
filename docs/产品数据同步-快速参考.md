# 产品数据同步 - 快速参考

## 🚀 三步同步产品数据

### 1️⃣ 编辑 CSV
打开 `产品数据表.csv`，修改价格或添加新产品，保存。

### 2️⃣ 预览变更
```bash
node scripts/sync-products.js --dry-run
```
查看将要进行的变更，确认无误。

### 3️⃣ 执行同步
```bash
node scripts/sync-products.js
```
完成同步！数据已更新到网站。

---

## 📋 CSV 格式

```csv
编号,品名名称,幅宽(cm),克重(g),成份,白色(元),彩色(元),空差,版布价
801#,涤天丝,150,55,90%天丝、10%涤,4.1,4.4,91,7
```

**重要提示：**
- 编号格式：`xxx#`（如 801#）
- 成分格式：`百分比%材料、百分比%材料`
- 价格可以写 `4.1` 或 `日/夜4.1`

---

## 🔄 常用操作

### 修改价格
1. 在 CSV 中修改对应产品的价格列
2. 运行 `node scripts/sync-products.js --dry-run` 预览
3. 确认后运行 `node scripts/sync-products.js` 同步

### 新增产品
1. 在 CSV 末尾添加新行，填写产品信息
2. 确保编号不重复（如：835#）
3. 运行同步脚本
4. 上传产品图片到 `public/images/products/编号_1.jpg`

### 批量调价
1. Excel 打开 CSV
2. 使用公式批量调整价格（如：`=原价*1.1` 涨价10%）
3. 保存后运行同步脚本

---

## ⚠️ 注意事项

- ✅ 同步前会自动备份原数据
- ✅ 新产品会自动分类（根据编号）
- ✅ 保留原有图片、标签等信息
- ❌ 不要在价格中包含 "元"、"¥" 等符号
- ❌ 小数点必须用 `.` 不能用 `,`
